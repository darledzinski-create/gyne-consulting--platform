# Dr. Dariusz Ledzinski – Online Gynaecological Consulting Platform

This repository hosts the source code for Dr. Ledzinski’s secure, web-based consultation platform.  
It’s built with Flask and deployed using Render.  

### How to run locally
1. Clone this repository
2. Run `pip install -r requirements.txt`
3. Start with `python app.py`
